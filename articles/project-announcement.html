Project Announcement

March 25, 2022

Today my fellow members and I are announcing the start of our full stack web application project. Our goal is to create an application for a restaurant that is easy to use and appealing to look at. We decided on this application because we wanted to help alleviate long wait times at restaurants for customers and allow employees to better manage the flow of customers.

Features

The features for this application will allow customers of a restaurant to make reservations. The restaurant employees will be able to access the reservations. We will create a database for all users to log in and access the appropriate information.
