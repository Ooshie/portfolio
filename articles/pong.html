Simple Pong

In this article I'll be describing my implementation of Pong. Firstly, I was required to create a web application that utilizes both Canvas and Local Storage. In order to create the canvas I used the canvas tag in my index.html file and defined its dimensions. In my JS file for the game, to render the canvas and everything in it I had to use window.onload and setInterval() to update the game's frames.



To actually draw everything visible I had to use fillStyle, fillRect, and fillText.



Implementing Local Storage was pretty straight forward when everything was set up. Initially I set my two variables equal to 0. Interestingly I don't have to use "var" to intialize the variable names.



Earlier I showed using fillText() allowed me to display the values of my Local Storage variables myScore and aiScore. To update the variables in the game.
To increment the local storage variables, it is pretty similar to incrementing regular variables. An article mentioned wrapping part of the statement in Number() to make sure it's a number.



The rest involved implementing the actual game of Pong where you can control your paddle with mouse movement. Canvas has addEventListener to recognize inputs like mouse movement shown in the first image. There is ball physics and and physics regarding making contact with the paddles the directions will change. It checks if the ball touches the ceilling, ground, paddles, and if it passes through a paddle's side it will increment the given score.



That concludes my brief explanation of my Simple Pong.
